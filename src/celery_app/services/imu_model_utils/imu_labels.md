HAR Labels based on imu (Sampling frequency: 25Hz, Window size = 2s (50 rows))

The model is based on https://github.com/yolish/har-with-imu-transformer

| Label | Activity          |
|-------|-------------------|
| 0     | Unknown           |
| 1     | Standing          |
| 2     | Sitting           |
| 3     | Lying             |
| 4     | Walking           |
| 5     | Climbing Stairs   |
| 6     | Running           |
